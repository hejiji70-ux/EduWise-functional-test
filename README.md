# EduWise 智课 — 功能测试

## 项目简介

本项目对 EduWise 智课在线教育平台进行功能测试，覆盖 Admin 后台、Teacher 讲师端、App 学员端三端核心业务流程。

## 测试范围

| 端 | 模块 | 用例数 |
|---|---|---|
| Admin 后台 | 登录、学员管理、讲师管理、课程管理、数据统计 | 53 |
| Teacher 讲师端 | 登录、课程管理、个人信息 | 38 |
| App 学员端 | 登录、首页、课程搜索、课程详情、订单 | 70 |
| 总计 | | 161 |

## 文件结构


## 有效 Bug 列表

| 编号 | 模块 | 描述 | 严重等级 |
|---|---|---|---|
| BUG-001 | Admin-学员管理 | 编辑学员密码校验缺失，可修改为纯数字短密码 | 中 |
| BUG-002 | Admin-学员管理 | 删除学员无二次确认 | 低 |
| BUG-003 | Admin-课程管理 | 课程分类选择后可取消选择，导致分类为空 | 中 |
| BUG-004 | Teacher-课程管理 | 创建课程封面图非必填，为空时展示异常 | 中 |
| BUG-005 | Teacher-课程管理 | 课程简介支持富文本，可插入外部图片链接 | 低 |
| BUG-006 | App-课程详情 | 课程价格为 0 时仍显示"立即购买"按钮 | 高 |
| BUG-007 | App-课程搜索 | 搜索结果为空时不显示提示信息 | 低 |
| BUG-008 | App-个人中心 | 我的订单中已完成订单不可删除 | 低 |
| BUG-009 | 通用-登录 | App 端连续输错密码无账号锁定机制 | 高 |

## 环境要求

- 后端服务：http://localhost:9096
- 浏览器：Chrome / Edge 最新版
- 测试方式：手工测试 + Bug 记录

## 使用说明

1. 打开 `test-cases/OnlineEdu_功能测试用例.xlsx` 查看完整测试用例
2. 打开 `bug-reports/EduWise智课-Bug.html` 查看 Bug 详情
3. 打开 `reports/EduWise智课-测试总结报告.md` 查看完整测试总结
